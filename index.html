Formas De Inserir o CSS no HTML

CSS Em linha(in-line)

Para inserir o css in-linne devemos colocar o atributo style na tag inicial!

CSS Interno(Internal CSS)

Para inserir o css interno devemos colocar a tag style dentro da tag head no HTML

CSS Externo(External CSS)

Para inserir o CSS externo devemos colocar a tag link no HEAD do HTMl e no atributo HREF="style.css"
